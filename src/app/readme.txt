En este ejercicio se pide:

  1. Reflejar en la variable compCount el número de componentes HelloComponent se están renderizando cada vez que yo aprieto el botón 'add'.

  2. Reflejar el numero de 'spans' que se están renderizando en cada componente, siempre se pintará la misma cantidad en cada componente. Ese es el valor que debe ser representado, la cantidad de spans que se ven en el componente HelloComponent.
